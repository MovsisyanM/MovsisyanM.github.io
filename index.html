"VioletRecon" desktop app
2019

This desktop application is written in C# and XAML, using only the .Net library, and the libraries previously made by me as APIs/drivers for various radio receivers, with no connection to the internet. This piece of software allows the user to automatically detect and record various radio signals, and in practice has been proven to be more than 900% efficient when compared to manual use with default software.

"Beeper" team schedule desktop app

.Net class library for signal direction finding via DFP1000B
2019

This is a class library written in C# without a network connection. The library provides a very simple way to work with and calculate signal direction data using two radio receivers. This library proved to be very useful since it allowed higher level software to calculate the orientation of the antennae without the use of a compass, determine where real sources of signals are compared to reflections from large terrain objects, determine the speed of a moving radio transmitter.

"Pewds vs Zombers" Videogame
2014 November

This is the first game I built using C#, UnityScript, and the Unity game engine. It was built for a 72-hour game jam, which is an event where thousands of game developers around the world challenge themselves by trying to make a fully-fledged game in a short amount of time around a specified theme. The theme was "fun to play, fun to watch" and it was around a YouTuber named PewDiePie who is still the biggest individual on the YouTube platform. I, at the time, had recently turned 14 and was already trying new things such as coding, 3D modeling, game design, so I made a decision that changed my life. Inspired by a young game developer called Nelson Sexton, I said to myself: "He is 16 and he made an enjoyable and popular game, why should I not be able to?" After seeing all the positive feedback from the players of my game and seeing that I liked coding, I started to learn software engineering.
